Competency Questions

Example Competency Questions

We present a set of competency questions which display how our the AIO captures the semantics of Data Sharing. We provide a sample SPARQL query for each question and an answer within our Use Case. We further provide versions of these competency questions within our aio-contracts repository in blockchain client code. A mapping of our two versions of the competency questions is shown below.

Competency Questions
Use Case-Specific Question RDF Triple Use Case-Specific Answer
Q1. What does Jeff intend to use Steven’s SNP Dataset for? aio:datarequest aio:intendeduse xsd:string Jeff intends to use the SNP profiles towards personalized drug therapy research for other types of cancer
Q2. Has Steven physically given the dataset to Jeff? aio:collaborationevent prov:activity aio:sharedata Steven has shared the Dataset with Jeff if the collaboration has an instance of Researcher Activity ’shareData’
Q3. Which publication by Jeff used the shared dataset? aio:researchartifact sio:‘is cited by’ some aio:publicationusingresearchartifact Steven’s research Artifact is cited by the new publication from Jef
Q4. How many researchers have cited Steven’s SNP dataset before? count(aio:researchartifact sio:’is cited by’ aio:publicationusingresearchartifact) 4 citations from other researchers to the SNP dataset
Q5. Why has Jeff given Steven a poor communication score in his review? aio:review aio:communicationscore xsd:integer Jeff iterates that Steven was non-communicative during the process
Q6. What factors go into Steven’s SCIENCEindex and why is it low? aio:researcher sio:’has attribute’ aio:hindex Jeff identifies that Steven’s SCIENCE-index is low due to a low h-index

SPARQL Queries

Question 1: What does the Data Seeker intend to use the data for?
We search for a specific instance of a Data Request Event via a given ?id which identifies the associated collaboration event
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?use
        WHERE {
            ?event a aio:CollaborationEvent .​
            ?event sio:SIO_000300 ?val .            //'has value'​
            FILTER(?val = ?id) .​
            ?event sio:SIO_000369  ?subevent .      //'has component  part'​
            ?subevent a aio:DataRequestEvent .
            ?subevent sio:SIO_000300 ?use .         //'has value'​
        }
Answer:
        "Jeff intends to use the SNP profiles towards personalized drug therapy research for othe types of cancer"
Question 2: Has the data exchanged hands between the collaborating researchers?
We search for a specific instance of a Data Sharing Event via a given ?id which identifies the associated collaboration event
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        ASK {
            ?event a aio:CollaborationEvent .​
            ?event sio:SIO_000300 ?val .            //'has value'​
            FILTER(?val = ?id) .​
            ?event sio:SIO_000369  ?subevent .      //'has component  part'​
            ?subevent a aio:DataSharingEvent .
            ?subevent sio:SIO_000300 ?bool .        //'has value'​
            FILTER(?bool) .
        }
Answer:
        "True"            //This represents that the data has been shared
Question 3: Which publication of a Data Seeker uses the shared Dataset?
Upon publication, the Data Seeker must report to the protocol the DOI of his publication. We can search for this via the specific Collaboration event.
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?doi 
        WHERE {
            ?event a aio:CollaborationEvent .
            ?event sio:SIO_000300 ?val .                    //'has value'​
            FILTER(?val = ?id) .
            ?event sio:SIO_000369  ?subevent .              //'has component  part'​
            ?subevent a aio:DataReuseReportingEvent .
            ?subevent sio:SIO_000312 ?publication .         //'has product'
            ?publication sio:SIO_000300 ?doi                //'has value'
        }
Answer:
        "doi:10.5281/01479"
Question 4: Does this publication properly cite the Dataset?
The Research artifact provided has an expected citation which we compare to the actual citation provided in the reported publication.
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        ASK { 
            ?event a aio:Collaboration_Event.​
            ?event sio:SIO_000300 ?val .                    //'has value'​
            FILTER(?val = ?id) .​
            ?event sio:SIO_000132 ?participant .            //'has participant'​
            ?participant sio:SIO_000064 ?artifact .         //'is provider of'​
            ?artifact sio:SIO_000008 ?expectcit .           //'has attribute'​
            ?expectcit sio:SIO_000300 ?cit1 .               //'has value'​
            ​
            ?event sio:SIO_000369  ?subevent .              //'has component part'​
            ?subevent a aio:Data_Reuse_Reporting_Event .​
            ?subevent sio:SIO_000312 ?publication .         //'has product'​
            ?publication sio:SIO_000028 ?actualcit .        //'has part'​
            ?actualcit sio:SIO_000300 ?cit2 .               //'has value'​
            
            FILTER(?cit1 = ?cit2). 
        }
Answer:
        "False"             //This represents that the Data Seeker has not properly cited the dataset and will be penalized.
Question 5: How many Researchers have cited my dataset listed on the protocol?
Query:
        PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?title ?citval
        WHERE {​
            ?etherobj a aio:Ethereum_Address .​
            ?etherobj sio:SIO_000300 ?etheradd .            //'has value'​
            FILTER(?etheradd = ?queryaddress) . ​
            ?etherobj sio:SIO_000674 ?researcher .          //'is unique identifier for'

            ?researcher sio:SIO_000064 ?artifact .          //'is provider of'
            ?artifact sio:SIO_000300 ?title .               //'has value'
            FILTER(?title = ?querytitle) .
            ?artifact sio:SIO_000300 ?citval .              //'has value'
            ?citval a xsd:integer .
        }
Answer:
        "SNP Profile Dataset" "4"
Question 6: Why has a researcher left a specific review?
We can query a specific review related to a collaboration event and it's natural language description
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?revdescription
        WHERE { 
            ?collabevent a aio:CollaborationEvent .
            ?collabevent sio:SIO_000300 ?val .              //'has value'​
            FILTER(?val = ?id) .​
            
            ?review a aio:DataSharerReview .
            ?review sio:SIO_001403 ?collabevent .           //'is associated with'
            ?review sio:SIO_000300 ?revdescription          //'has value'
        }
Answer:
        "Steven Hep provided decent quality data, but struggled to 
         communicate during the process and failed to answer 
         questions I had about the dataset"
Question 7: What factors are included in the SCIENCE-index?
The SCIENCE-index includes 5 statistics which represent a researcher's career contributions to science via publications and data sharing
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?type​
        WHERE { ​
            ?etherobj a aio:Ethereum_Address .​
            ?etherobj sio:SIO_000300 ?etheradd .            //'has value'​
            FILTER(?etheradd = ?queryaddress) . ​
            ?etherobj sio:SIO_000674 ?researcher .          //'is unique identifier for'

            ?researcher sio:SIO_000008 ?SCindex .           //'has attribute'​
            ?SCindex a aio:SCIENCE-index .​
            ?SCindex sio:SIO_000369 ?stats .                //'has component part'  ​
            ?stats a ?type .               
        }
Answer:
        "aio:CitationCount"
        "aio:PublicationCount"
        "aio:CollaboratorCount"
        "aio:h-index"
        "aio:DataSharingHistory"
