Competency Questions

Example Competency Questions

We present a set of competency questions which display how our the AIO captures the semantics of Data Sharing. We present two types of questions, a set geared towards handling data sharing, and a second geared towards incentivizing a participating researcher. We provide a sample SPARQL query for each question and an answer within our Use Case.

Data Handling Competency Questions
Competency Question Use Case-Specific Question Answer
Q1. What does the Data Seeker intend to use the data for? Q1. What does Jeff intend to use Steven's SNP Dataset for? aio:DataRequestEvent sio:'has value' some 'string' - Jeff intends to use the SNP profiles towards personalized drug therapy research for othe types of cancer
Q2. Has the data exchanged hands between the collaborating researchers? Q2. Has Steven physically given the dataset to Jeff ? aio:DataSharingEvent sio:‘has value' 'true' - Steven has shared the Dataset with Jeff
Q3. Which publication of a Data Seeker uses the shared Dataset? Q3. Which publication belonging to Jeff used the shared dataset? aio:PublicationUsingResearchArtifact sio:‘has value' some 'DOI' - The protocol stores the DOI of the reported publication from Jeff
Q4. Does this publication properly cite the Dataset? Q4. Has Jeff properly cited Steven's dataset? filter(aio:ExpectedCitation = aio:ActualCitation) - The actual citation contained in Jeff's publication matches the expected citation of Steven's dataset

SPARQL Queries

Question 1: What does the Data Seeker intend to use the data for?
We search for a specific instance of a Data Request Event via a given ?id which identifies the associated collaboration event
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?use
        WHERE {
            ?event a aio:CollaborationEvent .​
            ?event sio:SIO_000300 ?val .            //'has value'​
            FILTER(?val = ?id) .​
            ?event sio:SIO_000369  ?subevent .      //'has component  part'​
            ?subevent a aio:DataRequestEvent .
            ?subevent sio:SIO_000300 ?use .         //'has value'​
        }
Answer:
        Jeff intends to use the SNP profiles towards personalized drug therapy research for othe types of cancer
Question 2: Has the data exchanged hands between the collaborating researchers?
We search for a specific instance of a Data Sharing Event via a given ?id which identifies the associated collaboration event
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        ASK {
            ?event a aio:CollaborationEvent .​
            ?event sio:SIO_000300 ?val .            //'has value'​
            FILTER(?val = ?id) .​
            ?event sio:SIO_000369  ?subevent .      //'has component  part'​
            ?subevent a aio:DataSharingEvent .
            ?subevent sio:SIO_000300 ?bool .        //'has value'​
            FILTER(?bool) .
        }
Answer:
        True            //This represents that the data has been shared
Question 3: Which publication of a Data Seeker uses the shared Dataset?
Query:
        PREFIX sio: <http://semanticscience/org/resource/>
        PREFIX aio: <http://semanticweb.org/ontologies/2021/9/AIO#>
        SELECT ?doi 
        WHERE {
            ?event a aio:CollaborationEvent .
            ?event sio:SIO_000300 ?val .                    //'has value'​
            FILTER(?val = ?id) .
            ?event sio:SIO_000369  ?subevent .              //'has component  part'​
            ?subevent a aio:DataReuseReportingEvent .
            ?subevent sio:SIO_000312 ?publication .         //'has product'
            ?publication sio:SIO_000300 ?doi                //'has value'
        }
Answer:
        True            //This represents that the data has been shared
