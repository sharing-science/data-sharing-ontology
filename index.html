The Academic Incentive Ontology

About

The Data Sharing Ontology (DSO) is an OWL ontology which captures the difficult semantics of Blockchain-based data sharing. The DSO attempts to handle data-sharing while utilizing blockchain to implement an incentivize scheme to encourage peer-to-peer data sharing. The ontology has been developed as part of the ongoing SCIENCE project within Rensselaer Polytechnic Institute's Tetherless World Constellation and Institute for Data Exploration and Analysis.

Abstract

We present a framework for incentivized research data sharing using an ontology called the Data Sharing Ontology (DSO). The DSO captures the semantics of academic research data sharing and provides an operational specification for data sharing between researchers. The DSO includes a two-part incentive mechanism to confirm citations and reward reproducible research methods. The proposed solution is demonstrated using a dataset-sharing decentralized application use case.
